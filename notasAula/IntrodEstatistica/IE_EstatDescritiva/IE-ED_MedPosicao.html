Observe a quantidade de informação apresentada nessa tabela. Vamos apresentar, nessa seção, medidas que resumem um conjunto de dados por meio de valores que representam esses dados em termos de centralidade ou tendência comum. As cinco medidas que veremos são: média, mediana, moda, primeiro quartil e terceiro quartil.

Considere o conjunto numérico \(\{x_{1}, \dots, x_{n}\}\). A média desse conjunto é a soma de todos os seus valores dividida pela quantidade de elementos, isto é,

\[ \dfrac{x_{1} + \cdots + x_{n}}{n} \]

A média representa o valor que cada elemento do conjunto teria se todos os valores fossem iguais, mas a soma total permanecesse a mesma. É como redistribuir igualmente tudo o que temos, de forma que cada um fique com a mesma quantidade

Seja \(\{x_{1}, \dots, x_{n}\}\) um conjunto numérico ordenado de forma que \(x_{1} \le \cdots \le x_{n}\). A mediana é definida como sendo

\[ \left\{ \begin{array}{} \dfrac{x_{\frac{n}{2}} + x_{\frac{n}{2} + 1}}{2}, & \text{se } n \text{ é par}, \\ x_{\frac{n-1}{2}+1}, & \text{se } n \text{ é ímpar}. \end{array} \right. \]

Em outros termos, a mediana é o valor que divide o conjunto de dados ordenados em duas partes iguais, de modo que a metade dos valores está abaixo dela e a outra metade está acima. Em conjuntos com número ímpar de elementos, a mediana é o valor central. Em conjuntos com número par, a mediana é a média dos dois valores centrais.

A moda de um conjunto de dados é o valor - ou os valores - que ocorrem com maior frequência, desde que essa frequência seja superior a 1. Quando todos os valores aparecem apenas uma vez, dizemos que o conjunto não possui moda.

A moda representa o valor mais típico ou mais popular em um conjunto de dados - aquele que aparece mais vezes. Note que, diferentemente das outras medidas, a moda pode ser aplicada a variáveis qualitativas. Note também que um conjunto de dados pode ter mais de uma moda, caso dois ou mais valores apareçam com a mesma frequência, desde que essa frequência seja a maior entre todos os valores.

Seja \(\{x_{1}, \dots, x_{n}\}\) um conjunto numérico ordenado de forma que \(x_{1} \le \cdots \le x_{n}\). Dividimos esse conjunto em dois subconjuntos:

  • A metade inferior, que contém os elementos menores que a mediana;
  • A metade superior, que contém os elementos maiores que a mediana.
O primeiro quartil é a mediana da metade inferior, e o terceiro quartil é a mediana da metade superior.

Considere o conjunto \(\{2, 3, 4, 5, 5, 5, 7, 7, 8, 9\}\). A média, a mediana, a moda, o primeiro e o terceiro quartil são, respectivamente, 5.5, 5, 5, 4 e 7 (faça as contas).

No exemplo acima, é relativamente fácil calcular as medidas de posição, pois a quantidade de dados é pequena. Mas se considerarmos a tabela apresentada no início da seção? Por exemplo, qual é a média, a mediana e a moda da altura dos 50 alunos avaliados? Nesses casos, a utilização de software voltados para estatística é muito proveitoso.

Ao longo dessas notas, vou apresentar alguns comandos para se usar no software R. Por exemplo, você pode baixar o arquivo de extensão .csv que contém as informações da tabela do início da seção, clicando aqui. Depois, dentro de R, usar o comando 

dados <- read.csv("CAMINHO_COMPLETO/exemplo.csv", sep = ",")
para importar a tabela para o R.

Observação: no lugar de CAMINHO_COMPLETO, coloque o caminho onde o arquivo exemplo.csv está salvo no seu computador, como por exemplo:

"C:/Usuários/seunome/Documentos/..."
.

Depois disso, use os comandos 

Turma <- dados$Turma
Sexo <- dados$Sexo
Idade <- dados$Idade
Altura <- dados$Altura
Peso <- dados$Peso
Filho <- dados$Filho
Fumante <- dados$Fumante
Tolerante <- dados$Tolerante
Cinema <- dados$Cinema
OpCine <- dados$OpCine
TV <- dados$TV
OpTV <- dados$OpTV
para atribuir cada coluna a uma variável.

Feito isso, podemos usar o comando 

mean(Altura)
para determinar a média das alturas e 
median(Altura)
para determinar a mediana das alturas, obtendo, respectivamente, 1.667 e 1.655. Para a moda, é preciso usar os comandos 
library(modeest)
mfv(Altura)
Nesse caso, obtemos como resposta 1.6 e 1.7, ou seja, a variável "Altura" tem duas modas.

Por fim, para determinar o primeiro e o terceiro quartis basta fazer 

quartile(Altura)
e obter 1.6 e 1.7075, respectivamente.
