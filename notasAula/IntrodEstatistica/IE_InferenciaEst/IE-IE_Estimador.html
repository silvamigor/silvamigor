Introdução à estatística

Capítulo 3 - Inferência estatística

Seção 1 - Parâmetro, estimador e estimativa

Suponha que queiramos avaliar a média da altura da população de Belo Horizonte. Segundo o Censo de 2022, essa população totalizava 2.315.560 pessoas. Medir a altura de cada indivíduo dessa população e calcular a média seria uma tarefa inviável. No entanto, se selecionarmos uma parte representativa dessa população — por exemplo, 500 pessoas —, medir a altura de cada uma e calcular a média já se torna algo factível. Um subconjunto da população escolhido para fins de estudo é chamado de amostra.

A inferência estatística é a área da Estatística que desenvolve e aplica técnicas para estudar características da população com base em informações fornecidas por uma amostra. É importante destacar que diferentes amostras de uma mesma população podem produzir resultados distintos, devido ao caráter aleatório de sua seleção. Por isso, o estudo de amostras envolve conceitos e ferramentas da Teoria das Probabilidades.

Em Estatística, estudamos características dos indivíduos de uma população. Essas características são chamadas de variáveis estatísticas (ver Capítulo 1, Seção 1). Por exemplo, em um estudo sobre os moradores de Belo Horizonte, podemos estar interessados na altura de cada pessoa — nesse caso, a altura é a variável estatística.

Quando desejamos resumir numericamente uma característica da população inteira, usamos um parâmetro. Por exemplo, a média das alturas de todos os moradores de Belo Horizonte é um parâmetro. Os parâmetros são números fixos, mas geralmente desconhecidos, pois raramente conseguimos medir todos os indivíduos da população.

Como não podemos acessar toda a população, trabalhamos com uma amostra — um subconjunto representativo dos indivíduos. Os dados dessa amostra são tratados como realizações de variáveis aleatórias, já que a amostra é selecionada de maneira aleatória.

Uma função construída com os dados da amostra, usada para estimar um parâmetro da população, é chamada de estimador. Por exemplo, a média das alturas observadas na amostra é um estimador da média da população.

Quando aplicamos o estimador a uma amostra específica e obtemos um valor numérico, esse valor é chamado de estimativa.

Suponha que estejamos interessados em saber a média da altura dos alunos de certa escola. Esse valor, desconhecido, é o nosso parâmetro, denotado por \(\mu\). Para isso, escolhemos aleatoriamente 10 alunos e medimos a altura de cada um. Essa seleção constitui uma amostra de 10 alunos.

Note que diferentes amostras podem conter alunos distintos e, portanto, alturas diferentes. Estamos, assim, diante de uma sucessão de 10 experimentos aleatórios, em que, a cada experimento, escolhemos 1 aluno da escola. O espaço amostral de cada experimento é o conjunto \(\Omega\), formado por todos os alunos.

Podemos, então, definir a variável aleatória \(X_i : \Omega \to \mathbb{R}\), que representa a altura do aluno escolhido no \(i\)-ésimo experimento. Se somarmos as alturas dos alunos escolhidos e dividirmos por 10, obtemos a média amostral.

Pensando nisso, definimos o estimador:

\[ \widehat{\mu} = \dfrac{X_1 + \cdots + X_{10}}{10}. \]

Esse estimador fornece uma aproximação para a média populacional \(\mu\). Quando os valores específicos de \(X_1, \dots, X_{10}\) são observados, a média calculada constitui a estimativa da média da população, com base naquela amostra específica.

Um estimador \(\widehat{\lambda}\) para um parâmetro \(\lambda\) é não viciado, se \(E(\widehat{\lambda}) = \lambda\).

Um estimador \(\widehat{\lambda}_{n}\) para um parâmetro \(\lambda\), associado a uma amostra de tamanho \(n\), é consistente, se

\[ \lim\limits_{n \to \infty}E(\widehat{\lambda}_{n}) = \lambda \qquad \text{ e } \qquad \lim\limits_{n \to \infty}V(\widehat{\lambda}_{n}) = 0. \]

Suponha que certa característica de uma população tem média \(\mu\) e variância \(\sigma^{2}\). Tais valores, são desconhecidos. Para estimar a variância, considere uma amostra \((X_{1}, \dots, X_{n})\), onde cada \(X_{i}\) são variáveis aleatórias independentes e identicamente distribuídas, com \(E(X_{i}) = \mu\) e \(V(X_{i}) = \sigma^{2}\). Pode-se mostrar que \(E(\widehat{\mu}) = \mu\) e \(V(\widehat{\mu}) = \frac{\sigma^{2}}{n}\). Um estimador "natural" para a variância é

\[ \widehat{\sigma}^{2} = \dfrac{1}{n}\sum_{i=1}^{n}(X_{i} - \widehat{\mu})^{2}. \]

Note que

\[ \begin{split} \displaystyle\sum_{i=1}^{n}(X_{i} - \widehat{\mu})^{2} &= \sum_{i=1}^{n}(X_{i} - \mu + \mu - \widehat{\mu})^{2} = \sum_{i=1}^{n}\big((X_{i} - \mu)^{2} + 2(X_{i} - \mu)(\mu - \widehat{\mu}) + (\widehat{\mu} - \mu)^{2}\big) = \\ &= \sum_{i=1}^{n}(X_{i} - \mu)^{2} + 2\sum_{i=1}^{n}(\mu - \widehat{\mu})(X_{i} - \mu) + \sum_{i=1}^{n}(\widehat{\mu} - \mu)^{2} = \sum_{i=1}^{n}(X_{i} - \mu)^{2} + 2(\mu - \widehat{\mu})\sum_{i=1}^{n}(X_{i} - \mu) + n(\widehat{\mu} - \mu)^{2} = \\ &= \sum_{i=1}^{n}(X_{i} - \mu)^{2} + 2(\mu - \widehat{\mu})(n\widehat{\mu} - n\mu) + n(\widehat{\mu} - \mu)^{2} = \sum_{i=1}^{n}(X_{i} - \mu)^{2} - n(\widehat{\mu} - \mu)^{2}. \end{split} \]

Assim,

\[ \begin{split} E(\widehat{\sigma}^{2}) &= E\left(\dfrac{1}{n}\sum_{i=1}^{n}(X_{i} - \widehat{\mu})^{2}\right) = \dfrac{1}{n}E\left(\sum_{i=1}^{n}(X_{i} - \widehat{\mu})^{2}\right) = \\ &= \dfrac{1}{n}\left(E\big(\sum_{i=1}^{n}(X_{i} - \mu)^{2}\big) - nE\big(\widehat{\mu} - \mu)^{2}\right) = \dfrac{1}{n}\sum_{i=1}^{n}E\big((X_{i} - \mu)^{2}\big) - E\big(\widehat{\mu} - \mu)^{2}\big) = \\ &= \dfrac{1}{n}\sum_{i=1}^{n}V(X_{i}) - V(\widehat{\mu}) = \dfrac{1}{n}n\sigma^{2} - \dfrac{\sigma^{2}}{n} = \dfrac{n-1}{n}\sigma^{2}. \end{split} \]

Portanto, \(\widehat{\sigma}^{2}\) é um estimador viciado para \(\sigma^{2}\). Note que podemos eliminar o vício multiplicando \(\widehat{\sigma}^{2}\) por \(\frac{n}{n-1}\). Dessa forma, podemos usar um outro estimador para a variância, dito variância amostral (ver Capítulo 1, Seção 3), conforme abaixo

\[ S^{2} = \displaystyle\dfrac{1}{n-1}\sum_{i=1}^{n}(X_{i} - \widehat{\mu})^{2}. \]
