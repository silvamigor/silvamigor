Funções exponenciais

Aula 6 - Propriedades da função exponencial de base \(a\) — parte II

Vamos continuar com as propriedades da função \(f(x) = a^{x}\), com \(a \in \bbR_{+}^{\ast} - \{1\}\).

Sejam \(x, y \in \bbR\), com \(x < y\).

  1. Se \(1 < a\), então \(f(x) < f(y)\), ou seja, \(f\) é crescente.
  2. Se \(0 < a < 1\), então \(f(y) < f(x)\), ou seja, \(f\) é decrescente.
Demonstração
Vamos dar uma ideia da demonstração de (a). Vimos no Lema 2 - Aula 1, que tal propriedade vale para os naturais. Para os inteiros, vemos que se \(1 < a\), então multiplicando por \(a^{-1}\), temos que \(a^{-1} < 1\). Seguindo assim, temos que \(a^{m} < a^{n}\), se \(m < n\), com \(m,n \in \bbZ\). Para os racionais e os irracionais, omitiremos a demonstração.

A função \(f\) é injetiva.

Demonstração
Sejam \(x_{1}, x_{2} \in \bbR\), com \(x_{1} \neq x_{2}\). Então \(x_{1} < x_{2}\) ou \(x_{2} < x_{1}\). Assuma, sem perda de generalidade, que \(x_{1} < x_{2}\). Pela Proposição 2, temos que \(f(x_{1}) < f(x_{2})\), se \(a > 1\), ou \(f(x_{2}) < f(x_{1})\), se \(0 < a < 1\). De todo modo, \(f(x_{1}) \neq f(x_{2})\).

A função \(f\) é sobrejetiva.

Demonstração
Essa demonstração também será omitida, por envolver conceitos não apresentados nessas notas.
