Como mostrar que $1 < 2$ em $\mathbb{N}$?

Essa pergunta pode parece estranha e até meio tola, se levarmos em consideração a ideia de que os números naturais foram criados para atender a necessidade de contar, ou seja, os números naturais, intuitivamente, são entidades abstratas, que nos servem de modelo para contar quantidades de uma determinada grandeza discreta. Por exemplo, se você tem uma quantidade de maçãs e quer saber quantas são, você faz corresponder uma maçã ao número $1$, uma outra maçã diferente ao número $2$, e assim sucessivamente, até a última maçã. O número natural associado a essa última maça é a quantidade de maçãs que você tem. Digamos que o número encontrado tenha sido $6$.

Os números naturais, da maneira como entendemos, carregam a ideia de que todo número tem um sucessor, quer dizer, um número que vem logo depois dele. O sucessor de um número e os sucessores consecutivos dos sucessores de um número, são maiores que o número dado. Por exemplo, $2$ é o sucessor de $1$, logo $1 < 2$; $3$ é o sucessor de $2$, logo $2 < 3$ e $1 < 3$. Disso fica claro a verdade da desigualdade que queremos mostrar. Contudo, note que usamos apenas noções intuitivas do que são os números naturais. Para um primeiro contato com esses objetos isso é o suficiente. Acontece que na matemática, precisamos tornar as coisas mais formais, pois só assim conseguimos provar com precisão as afirmações. O método que formaliza as teorias matemáticas é chamado de sistema axiomático. Um sistema axiomático é uma construção constituída por:

  • axiomas, isto é, enunciados aceitos sem demonstrações;
  • teoremas, isto é, enunciados demonstrados a partir dos axiomas;
  • conceitos primitivos, isto é, conceitos admitidos sem definição;
  • conceitos derivados, isto é, conceitos definidos a partir dos conceitos primitivos.

A ideia por trás de um sistema axiomático é utilizar os conceitos primitivos para definir os conceitos derivados e usar os axiomas para provar os teoremas. Assim evitamos a regressão ao infinito e a circularidade. Pense só! Se você quer definir uma certa palavra, você vai precisar usar outras palavras diferentes para isso. Cada palavra usada nessa definição precisa de outras para serem definidas. Por exemplo, se definirmos plano como sendo o conjunto em que os pontos são seus elementos e as retas são seus subconjuntos, estamos usando as palavras conjunto, ponto, elemento, retas e subconjunto. Cada uma dessas palavras devem ser definidas para entendermos por completo o que é um plano. Dessa forma, caminhamos indefinidamente, sem fim, e não definimos nada, ou caímos em uma circularidade, isto é, uma palavra define uma outra, que defini outra, até voltarmos à primeira.

O mesmo acontece com teoremas. Para demonstrarmos um teorema precisamos usar fatos que já sabemos serem verdadeiros. Essas afirmações que já conhecemos a veracidade delas foram provadas usando outras afirmações verdadeiras. Daí, seguimos sem fim ou caímos, outra vez, em uma circularidade. Para evitar esse problema, precisamos usar coisas que funcionam como barreiras, quer dizer, coisas que ao recuarmos o raciocínio, chegamos a um limite. Essas coisas são:

  • para as definições, os conceitos primitivos,
  • para os teoremas, os axiomas.

A pessoa que apresenta um sistema axiomático é que escolhe quem serão os conceitos primitivos e os axiomas. Normalmente, prefere-se aqueles axiomas e conceitos com viés intuitivo, porém a quem prefira àqueles axiomas e conceitos que tornem o sistema mais simples, isto é, o menor número de axiomas e conceitos que é capaz de provar o máximo de teoremas e definir o máximo de conceitos. Frequentemente, em um sistema axiomático, os teoremas recebem outros nomes, dependendo da importância do resultado ou de sua utilização ou consequência em outros resultados. Assim, os teoremas também recebem os nomes: proposição, lema e corolário.

O objetivo dessa seção é apresentar o sistema axiomático usado para descrever concisa e precisamente o conjunto dos números naturais, devida ao matemático italiano Giuseppe Peano (1858-1932) e a partir dele provar que $1 < 2$. Para isso, precisamos conhecer quem são os conceitos primitivos e quem são os axiomas. Os conceitos primitivos são "conjunto" e "função", ou seja, nesse sistema axiomático, não se explica o que é um conjunto, nem o que é uma função. Os axiomas, ditos axiomas de Peano, são:

  1. existe um conjunto $\mathbb{N}$ e uma função $s: \mathbb{N} \to \mathbb{N}$ tais que
    1. $s$ é injetora;
    2. existe um elemento em $\mathbb{N}$, denotado por $1$, que não pertence a imagem de $s$, isto é, $1 \not\in \text{im}(s)$;
  2. se $X \subseteq \mathbb{N}$ é um subconjunto satisfazendo
    1. $1 \in X$,
    2. se $k \in X$, então $s(k) \in X$,
    então $X = \mathbb{N}$.

O conjunto $\mathbb{N}$ é o que chamamos de conjunto dos números naturais e seus elementos são ditos, números naturais (ou, simplesmente, naturais). A letra $s$ que nomeia a função do Axioma 1, vem da palavra sucessor. Assim, dado $n \in \mathbb{N}$, chamamos $s(n)$ de sucessor de $x$. Dizer, no Axioma 1 (a) que $s$ é injetiva, significa dizer que se dois elementos de $\mathbb{N}$ são diferentes, então suas imagens são diferentes. Em outras palavras, quer dizer que dois números naturais diferentes tem sucessores diferentes. Com esse linguajar, o Axioma 1 (b), diz para nós que o $1$ não é sucessor de nenhum natural. O Axioma 2 é chamado de Princípio da Indução Matemática.

Defina $2$ como sendo $s(1)$, $3$ como sendo $s\big(s(1)\big) = s(2)$, e assim sucessivamente. Como todos definidos acima pertencem a $\mathbb{N}$, temos que $\{1, s(1), s\big(s(1)\big), \cdots\} =$ $\{1, 2, 3, \cdots\} \subseteq \mathbb{N}$. A pergunta que se coloca é: $\mathbb{N}$ contém outros elementos? A resposta é não, pois como $1 \in \{1, 2, 3, \cdots\}$ e $\{1, 2, 3, \cdots\}$ foi construído de forma a conter o sucessor de qualquer elemento nele contido, temos, pelo Princípio da Indução Matemática, que $\mathbb{N} = \{1, 2, 3, \cdots\}$. Isso mostra que o sistema axiomático proposto por Peano formaliza a ideia intuitiva que temos do conjunto dos números naturais.

Para concluir essa seção e mostrar que $1 < 2$, precisamos definir uma maneira de comparar os elementos de $\mathbb{N}$, e daí concluir que $2$ é maior do que $1$. Para isso precisamos definir a adição em $\mathbb{N}$. A adição entre dois números naturais, $m$ e $n$, denotada por $m + n$ é definida recursivamente da seguinte maneira:

\[ \begin{array}{rcl} m + 1 & = & s(m) \\ m + s(n) & = & s(m + n) \end{array} \]

Observe que podemos dispensar a notação $s(n)$ para indicar o sucessor de $m$, para usar $n + 1$. Assim, por um lado, $m + s(n) = m + (n + 1)$, e por outro, $s(m + n) = (m + n) + 1$, ou seja, $m + (n + 1) = (m + n) + 1$. Isso indica que a adição goza da propriedade associativa. De fato, isso é verdade. Usando o Princípio da Indução Matemática, podemos mostrar que a adição é:

  • associativa, isto é, dados $m, n, p \in \mathbb{N}$, temos que $(m + n) + p = m + (n + p)$;
  • comutatividade, isto é, dados $m, n \in \mathbb{N}$, temos que $m + n = n + n$.

Agora, estamos prontos para definir uma relação de ordem entre os números naturais. Ela será feita em termos da adição definida anteriormente. Dados $m$ e $n$ dois números naturais, dizemos que $m$ é menor do que $n$ e escrevemos $m < n$, se existe $p \in \mathbb{N}$ tal que $n = m + p$. Portanto, uma vez que $2 = s(1) = 1 + 1$, temos que $1 < 2$.
